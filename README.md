# Homework Six
## This is the third through twelfth page of travel-fly, tours and each tour page
#### Purpose 
This homework assignment will:
- Create multiple pages from a CSS class 
- Make a CSS template
- Teach you how to make your site responsive
- Teach you how to deal with images next to text 
- Use media queries
- Use percentages
 
#### Task 
This homework assignment will be the about page from the travel-fly prototype
- Create a project named homework-six.
- Tour page must look like the prototype. Text must look good inside each image.
- Each individual page must look like the prototype and the text inside the header image must stay in place and the rest of the page must be responsive.
- Page must be responsive. 
- Elements can be aligned as you wish, but they must look good and not touch the sides of the browser.
 

#### Criteria for Success 
Assignment will be graded by the rubric that is available through Canvas.
